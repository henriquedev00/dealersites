# DealerSites

##### O projeto requer no mínimo a versão 8.0.2 do PHP.

- Rodar o comando:
    ```sh
    cp .env.example .env
    ```
- Configurar os dados de acesso do banco de dados no .env
- Rodar o comando:
    ```sh
    composer install
    ```
- Rodar o comando:
    ```sh
    php artisan key:generate
    ```
- Rodar o comando:
    ```sh
    npm install
    ```
- Rodar o comando:
    ```sh
    npm run dev
    ```
- Rodar o comando:
    ```sh
    php artisan migrate --seed
    ```
